# uTools 歇后语大全

> 收录 14000+ 条歇后语，支持模糊搜索的 uTools 插件

## 功能特性

- 🔍 **模糊搜索** - 支持按谜面、谜底、释义搜索
- 📄 **分页展示** - 支持 10/20/30/50/100 条每页
- ⌨️ **键盘导航** - 上下键选择，PageUp/PageDown 翻页
- 🎨 **中国风 UI** - 宣纸背景、朱砂红点缀

## 安装方法

1. 下载本仓库 `public` 文件夹
2. 打开 uTools 开发者工具
3. 选择 `public` 文件夹加载插件

## 使用说明

- 在 uTools 中输入「歇后语」即可打开插件
- 输入关键词进行模糊搜索
- 使用上下键或鼠标选择查看详情
